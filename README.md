# `sparky` - LLM Cluster Manager for DGX Spark

`sparky` primarily aims to automagically manage the presence of LLMs running on a spark cluster by loading the models as-needed, watching for crashes, managing evictions, etc.

Check out the obsidian vault in `0-sparky-docs` (named to influence alphabetical sort) for details. Observe `Contribution.md` and `Code Style.md`, bare minimum, before contributing.

`sparky` is a work-in-progress and mostly a skeleton desiring flesh at this moment. It is not usable yet.
